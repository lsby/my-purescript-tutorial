## 疑难杂症

```
- 如果安装包或编译时出现权限问题, 一般的原因可能是文件操作数量限制. 请尽可能关闭其他程序, 并使用-j参数限制线程数量.
  - 比如spago build -j 1.
```

## 常用命令

```
进入交互模式
spago repl

初始化项目
spago init

编译
spago build

运行
spago run

测试
spago test

打包为可执行程序
spago bundle-app

打包为模块
spago bundle-module

安装依赖
spago install <包名称>
```
