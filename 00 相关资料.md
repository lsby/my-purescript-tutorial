## 常用命令

```
进入交互模式
spago repl

初始化项目
spago init

编译
spago build

运行
spago run

测试
spago test

打包为可执行程序
spago bundle-app

打包为模块
spago bundle-module

安装依赖
spago install <包名称>
```
