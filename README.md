# PureScript 教程

这是一份`PureScript`教程.

期待读者有一定的编程基础, 理解工程中的常见概念, 例如包, 包管理器, 运行时等.

但不需要有面向对象和函数式的经验.

本教程只做为入门教程, 旨在简单快速带读者入门.

所以本教程不会纠结细节, 不会讲太复杂的东西, 当读者遇到本教程没有涉及到的东西时, 需要自行查资料, 提问等.

也欢迎与我讨论, 有任何意见建议欢迎联系.
